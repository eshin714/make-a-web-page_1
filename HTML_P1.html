STRUCTURE AND STYLE

HTML's TREE LIKE STRUCTURE

HTML is made up of a "tree" like structure. You can have HTML "branches" or elements inside other HTML "branches" or elements. In other words you can have boxes within boxes within boxes! The DOM or Document Object Model is the the tree like structure that translates the structure.

TAGS and DOMS

HTML is the language of the internet. It uses "tags" to decide what it will do with the text being used. The DOM then translates that "tag" and displays it on the page. These "tags" can be MANY things. Each tag can also have attributes with values. These values can change the elements even more.

HTML and BOXES

Everything on a webpage is a BOX. Thinking of things in boxes will help build webpages. It helps rearrange things to give uses a different experience.

CSS OR CASCADING STYLE SHEETS

Cascading style sheets or CSS alls you to use a special syntax and special rules to change the STYLE of the page. CSS will create the look and feel of the page and give it a unique style that will make the page feel original and aesthetically pleasing.

INHERITANCE

Inheritance involves the HIGHEST to LOWEST level of applications to the style. At the highest level the rule or syntax will apply to the lower, but the syntax or rule at the lowest level will not apply to the ones above it. It relies on the document tree or the hierarchy of the HTML elements. It should be noted that in general, the descendent elements will inherit text related properties, but not box-related properties.

AVOID REPETITION

The use of CSS can help avoid repeating yourself! With the styles you can "class"-ify titles and headers. You can also set colors and fonts for similar sections as you see fit. With CSS its easy to avoid repetition and be more efficient with time and adding more code which could mean more errors. It also helps your code become more readable. If you're consistent in your code people can fix and change things accordingly and efficiently. It also helps when you're able to see where you have made mistakes without all the repetition cluttering your code. With CSS you can create new sections, titles, change background colors, fonts, and many other things!
As Programmers and Coders we should avoid repetition!
